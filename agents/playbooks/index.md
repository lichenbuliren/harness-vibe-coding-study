# Agent Playbooks / Agent 操作手册

本目录将存储可复用的执行手册。

This directory will store reusable execution playbooks.

手册应描述何时使用某个工作流、需要什么输入、遵循哪些步骤、必须产出什么证据、以及结果应记录在哪里。

Playbooks should describe when to use a workflow, what inputs it needs, what
steps it follows, what evidence it must produce, and where outcomes should be
recorded.

## Playbooks / 操作手册

- `subagent-delegation.md`：主 agent 委派有界子任务并将结果整合回项目主线的操作流程。
  Main agent workflow for delegating bounded subtask work and integrating the
  result back into the project mainline.
