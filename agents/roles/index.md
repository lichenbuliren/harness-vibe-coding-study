# Agent Roles / Agent 角色

本目录将存储 agent 工作的角色定义。

This directory will store role definitions for agent work.

角色文档应定义职责、权限、预期产出、交接行为和验证期望。

Role documents should define responsibility, authority, expected outputs,
handoff behavior, and verification expectations.

## Role Sets / 角色集

- `default-agent-roles.md`：面向 agent-first 项目的推荐最小角色集，包括主 agent、需求/PRD agent、编码 agent、设计评审 agent、代码评审 agent、验证 agent 和文档 agent。
  Recommended minimum role set for agent-first projects, including main agent,
  requirements/PRD agent, coding agent, design review agent, code review agent,
  verification agent, and documentation agent.
