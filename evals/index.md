# Evaluations / 评估

`evals/` 定义如何评判质量。

`evals/` defines how quality is judged.

使用本目录存放可在不同实验和 lab 工作中复用的评分标准、检查清单和关卡规则。

Use this directory for rubrics, checklists, and gates that can be reused across
experiments and lab work.

## Sections / 分区

- `rubrics/`：定性和基于证据的评分标准。
  Qualitative and evidence-based scoring standards.
- `checklists/`：可重复的通过/不通过检查项和完成检查。
  Repeatable pass/fail gates and completion checks.
