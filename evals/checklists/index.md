# Checklists / 检查清单

本目录将存储可重复的完成和质量检查清单。

This directory will store repeatable completion and quality checklists.

检查清单应足够具体，以便 agent 在声明工作完成之前执行。

Checklists should be concrete enough for agents to run before claiming work is
complete.

## Current Checklists / 当前检查清单

- `mobile-ui-state-feedback.md`：检查移动端 UI 操作是否清晰且可测试地暴露禁用态和就绪态。
  Checks that mobile UI actions expose disabled and ready states clearly and
  testably.
- `mobile-interaction-performance.md`：检查移动端交互是否保持状态本地化、避免不必要的重渲染，并验证面向用户的访问路径。
  Checks that mobile interactions keep state local, avoid avoidable rerenders,
  and verify the user-facing access path.
