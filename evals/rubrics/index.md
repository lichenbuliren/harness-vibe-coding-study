# Rubrics / 评分标准

本目录将存储评估评分标准。

This directory will store evaluation rubrics.

评分标准应定义被评判的质量、评分等级、强输出和弱输出的示例，以及每项评判所需的证据。

Rubrics should define the qualities being judged, scoring levels, examples of
strong and weak outputs, and evidence required for each judgment.

## Current Rubrics / 当前评分标准

- `harness-validation-rubric.md`：评估项目 harness 是否改进了上下文使用、任务构建、执行、验证、可恢复性、演进捕获和安全性。
  Evaluates whether the project harness improves context use, task framing,
  execution, verification, recoverability, evolution capture, and safety.
