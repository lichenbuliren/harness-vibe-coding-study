# Task Samples / 任务样本

本目录将存储用于测试 agent 工作流和 harness 行为的任务。

This directory will store tasks used to test agent workflows and harness
behavior.

任务样本应定义目标、起始上下文、约束、预期产物和评估钩子。

A task sample should define goal, starting context, constraints, expected
artifacts, and evaluation hooks.

## Current Task Samples / 当前任务样本

- `agent-first-project-tasks.md`：用于测试目录契约、演进更新、验证纪律、运行记录和源码合成的本地小型评估任务。
  Local mini-eval tasks for testing directory contracts, evolution updates,
  verification discipline, run records, and source synthesis.
