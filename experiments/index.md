# Experiments / 实验

`experiments/` 记录我们尝试了什么、在什么条件下、以及结果如何。

`experiments/` records what we tried, under what conditions, and what happened.

使用实验记录任务样本、运行报告、对比分析和回顾总结。把评估标准放在 `evals/` 中。

Use experiments for task samples, run reports, comparisons, and retrospectives.
Keep evaluation criteria in `evals/`.

## Sections / 分区

- `task-samples/`：用于测试 harness 行为的可复用任务提示和预期产物。
  Reusable task prompts and expected artifacts for testing harness behavior.
- `reports/`：记录条件、结果和经验教训的实验报告。
  Experiment reports that record conditions, outcomes, and lessons.
