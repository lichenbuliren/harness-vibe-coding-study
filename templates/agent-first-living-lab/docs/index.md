# Documentation Index / 文档索引

`docs/` 存储本 Agent-First Living Lab 的方法论和知识。

`docs/` stores methodology and knowledge for this Agent-First Living Lab.

## Sections / 分区

- `principles/`：稳定的原则和约束。 / Stable principles and constraints.
- `workflows/`：可重复的工作流程。 / Repeatable work processes.
- `patterns/`：实践中发现的可复用模式。 / Reusable patterns discovered through practice.
- `standards/`：跨领域项目标准。 / Cross-cutting project standards.
- `tools/`：工具说明与安装指导。 / Tool notes and installation guidance.
- `evolution/`：阶段级别的叙事记录和可分享的经验。 / Stage-level narrative records and shareable lessons.

## Placement Rule / 放置规则

- 将持久信念放在 `principles/`。 / Put durable beliefs in `principles/`.
- 将分步流程放在 `workflows/`。 / Put step-by-step procedures in `workflows/`.
- 将可复用方案放在 `patterns/`。 / Put reusable solutions in `patterns/`.
- 将跨领域强制性规则放在 `standards/`。 / Put mandatory cross-cutting rules in `standards/`.
- 将工具相关说明放在 `tools/`。 / Put tool-specific notes in `tools/`.
- 将阶段级回顾和成果放在 `evolution/`。 / Put stage-level retrospection and outcomes in `evolution/`.
