# Templates / 模板

`templates/` 存储从已验证的项目证据中提取的可复用骨架。

`templates/` stores reusable skeletons extracted from validated project
evidence.

这些模板应保持精简。不要将项目特定的 lab 历史、运行时状态、本地路径或已完成的演进条目复制到模板核心中。

These templates should be conservative. Do not copy project-specific lab
history, runtime state, local paths, or completed evolution entries into a
template core.
